Disaster Detection at the Edge

This is the IOT part of the project. It aims to bring disaster detection and early warning system to the edge from the cloud while staying connected to the cloud.
This consists of a distributed IOT based solution for earthquake detection and early warning system.
Using Raspberry Pi as a central IOT Hub for multiple IOT solutions and Node MCU units acting as slave boards deployed in different rooms of a given building providing real time data to the central hub through API calls and also capable of handling both security and automation in those rooms.
- The central hub is connected to Azure through REST API calls to provide a dashboard monitoring system and as an early warning system.

The central hub is responsible for following safety protocols such as

  • Kill the power, water and gas supply to prevent further hazard to people.
  • Sound based warning system.

Skin Disease Diagnosis

  • Using Custom Vision service of Azure cognitive services to predict different type of skin diseases which may arise during a flood scenario.
  • The accompanying Android App has the exported tensorflow model from customvision.ai to predict diseases offline.
  • It Eliminates the need Medical Test when there is lack of infrastructure during a calamity.
  • Provide faster medical diagnosis
